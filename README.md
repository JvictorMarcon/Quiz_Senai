# Quiz Senai

Um aplicativo web de quiz desenvolvido com Flask.

## Funcionalidades

- Interface interativa de quiz
- Sistema de ranking
- Design responsivo

## Requisitos

- Python 3.x
- Flask

## Como executar

```bash
python app.py
```

Acesse `http://localhost:5000` no seu navegador.

## Estrutura do projeto

- `app.py` - Aplicação principal
- `ranking.json` - Dados de ranking
- `static/` - Arquivos CSS
- `templates/` - Templates HTML
